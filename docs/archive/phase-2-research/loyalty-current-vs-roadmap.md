# Loyalty: Current State vs Roadmap

Last updated: 2026-03-03

## Short answer

1. **Yes, ROLLER has loyalty-related features now** (mainly memberships, benefits, and member self-service).
2. **A native points-based loyalty program is roadmap/planned**, not confirmed as generally available today.

## What exists now (from current docs)

1. Membership products (fixed and recurring).
2. Membership perks/benefits and redemption controls.
3. POS membership redemption and remaining-benefits visibility.
4. Online account self-service for recurring memberships (upgrade/renew/cancel, when configured).
5. Group membership product is in beta.

This means "loyalty" exists today mostly as **membership + benefits + redemption**.

## What roadmap says is coming

From the local ROLLER roadmap PDF (page 7):

1. Theme includes **Native Loyalty Program**.
2. It describes a **points-based loyalty platform native within ROLLER**.
3. It says it will use payment/card data to enrich profiles and track transactions/spending patterns.

## Gap between today and roadmap

1. **Today:** membership-led loyalty (discounts/perks/redemptions).
2. **Roadmap:** native points engine across transactions.
3. **Conclusion:** points-based native loyalty appears to be an upcoming capability, not a currently documented standard feature.

## What is uncertain

1. Exact release date and rollout scope of native loyalty.
2. Which plans/regions/tenants get first access.
3. Migration path from existing membership loyalty flows to points-based model.
