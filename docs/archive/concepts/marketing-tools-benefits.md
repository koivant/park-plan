# Marketing Tools Benefits (Concept)

Last updated: 2026-03-04

## What a marketing platform gives us

1. Recover lost sales:
   - Send reminders to guests who started checkout but did not complete payment.
2. Save team time:
   - Automate messages like welcome, birthday, post-visit, and win-back campaigns.
3. Better targeting:
   - Send different messages to different groups (families, teens, members, inactive guests).
4. More repeat visits:
   - Use visit and purchase history to send relevant offers.
5. Better loyalty adoption:
   - Promote loyalty sign-up, point/stamp progress, and reward usage.
6. Better reporting:
   - See which campaigns drive bookings, revenue, and return visits.
7. Multi-channel communication:
   - Manage email, SMS, and in some markets WhatsApp from one workflow.
8. Faster improvement:
   - A/B test messages, offers, and send times to improve conversion.

## Why this matters for park

1. More ticket revenue from existing website traffic.
2. Better guest retention between visits.
3. Stronger connection between checkout, loyalty, and marketing.
