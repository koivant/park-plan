# Platform Boundaries

Last updated: 2026-03-04

## WordPress handles

1. Country and park content pages.
2. Campaign pages and SEO.
3. Buy buttons that open ROLLER overlay checkout.
4. Account links and campaign URL parameters (UTM = Urchin Tracking Module tags).

## ROLLER handles

1. Checkout and payment processing.
2. Booking and transaction records.
3. Guest account and membership features.
4. POS redemption operations.

## External loyalty partner handles (if Option B is chosen)

1. Stamp/points history and balance logic.
2. Reward creation when stamp threshold is reached.
3. Reward states (`issued`, `redeemed`, `void`).
4. Notification trigger logic.
5. Reconciliation of missing events.
