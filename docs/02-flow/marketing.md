# Marketing (Flow)

Last updated: 2026-03-06

Q: What flow do we need from marketing platform features?
A:
1. Abandoned-cart recovery flow.
2. Lifecycle automation flow.
3. Segmented messaging flow.
4. Reporting flow for campaign performance.

Q: What is abandoned-cart recovery flow?
A:
1. Guest starts checkout.
2. No purchase after delay.
3. If contact + consent exist, send recovery message.
4. Guest returns using link when supported.

Q: What is lifecycle automation flow?
A:
1. Guest action happens (for example purchase or visit).
2. System places guest into correct journey stage.
3. Automated message is sent at the right time (pre-visit, post-visit, win-back).
4. Guest response updates next message timing or segment.

Q: What is segmented messaging flow?
A:
1. Guest data is grouped into segments (for example new, active, inactive, loyalty member).
2. Each segment gets different message content and offer logic.
3. Message channel is chosen based on available contact and consent.
4. Segment membership updates automatically as behavior changes.

Q: What is reporting flow for campaign performance?
A:
1. Campaign sends are logged.
2. Opens/clicks/visits/purchases are tracked.
3. Results are matched to campaign and segment.
4. Team reviews conversion and revenue impact and adjusts campaigns.
