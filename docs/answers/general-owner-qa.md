# General Owner Q&A

Last updated: 2026-03-26

Q: What checkout path should we use now?
A: Hosted ROLLER overlay checkout.
Sources:
1. https://mysupport.roller.software/hc/en-us/articles/5517737900687-Create-your-online-checkout
2. https://mysupport.roller.software/hc/en-us/articles/5494935998351-Share-your-online-checkout

Q: Do we need custom checkout now?
A: No for current scope. Build custom only for requirements hosted checkout cannot support.
Sources:
1. https://mysupport.roller.software/hc/en-us/articles/5494935998351-Share-your-online-checkout
2. https://mysupport.roller.software/hc/en-us/articles/9657999631759-Add-the-overlay-checkout-experience-to-your-website

Q: Is ROLLER Payments mandatory?
A: Yes for progressive checkout.
Sources:
1. https://mysupport.roller.software/hc/en-us/articles/5517737900687-Create-your-online-checkout

Q: What is current loyalty baseline?
A: Use ROLLER account identity and evaluate Patch-first interim loyalty while native ROLLER points remain unconfirmed.
Sources:
1. https://mysupport.roller.software/hc/en-us/articles/7382787636879-Sign-in-and-use-online-accounts
2. https://mysupport.roller.software/hc/en-us/articles/12557902973839-ROLLER-Patch-Retention
3. ../sources/internal-evidence.md#ie-001-roller-support-note-2026-03-03

Q: What are the key risks right now?
A:
1. Native loyalty timeline/scope is not locked publicly.
2. Pay-later in progressive overlay has conflicting documentation.
3. Patch stamp-style UI fit is not fully confirmed for this account model.
4. Migration from Patch to native loyalty is not documented publicly.
Sources:
1. ../sources/internal-evidence.md#ie-001-roller-support-note-2026-03-03
2. https://mysupport.roller.software/hc/en-us/articles/6052170934543-Allow-your-guests-to-book-online-without-paying
3. https://mysupport.roller.software/hc/en-us/articles/360001841316-Add-online-payment-methods-with-ROLLER-Payments

5. https://mysupport.roller.software/hc/en-us/articles/12557902973839-ROLLER-Patch-Retention
