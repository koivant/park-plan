# Marketing (Concept)

Last updated: 2026-03-11

Q: What features do we need from a marketing platform?
A:
- Abandoned-cart recovery.
- Automated lifecycle messages (pre-visit, post-visit, win-back).
- Segmentation by customer behavior.
- Basic campaign performance tracking.

Q: Which service could be the best fit?
A: Start with Patch in evaluation because it has official ROLLER integration and attraction-focused setup. Keep Klaviyo and Campaign Monitor as alternatives based on channel needs. Note: Patch and Klaviyo integrations require API add-ons in ROLLER. Final "best fit" is verification needed until pilot KPI results are compared.

Q: Does abandoned-cart recovery need extra technical setup?
A: Yes. For Patch + ROLLER, setup includes GTM configuration and installing/configuring Patch tracking tags/scripts per location.
Source: https://help.patchretention.com/how-to-set-up-the-roller-abandoned-cart-automation

Q: How should we ideally monetise peak times and seasons?
A: Use time-based pricing rules: raise prices when demand/capacity is constrained (peak periods) and use lower-friction offers in off-peak windows. Then support off-peak demand with automated lifecycle campaigns.
